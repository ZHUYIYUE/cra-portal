# CRA Portal 云端部署指南

> 预计时间：5分钟 | 难度：⭐（很简单）

## 🎯 部署目标

将CRA Portal部署到 **Render云端**，获得公网访问地址，随时随地用手机/电脑访问。

**部署后效果：**
- ✅ 公网访问（`https://你的项目名.onrender.com`）
- ✅ 手机随时访问
- ✅ 数据自动同步（网站 ↔ 微信对话）

---

## 📋 部署步骤

### 第1步：创建GitHub仓库（1分钟）

1. 访问 [GitHub.com](https://github.com) 并登录
2. 点击右上角 **"+"** → **"New repository"**
3. 填写信息：
   - **Repository name**: `cra-portal`（或你喜欢的名字）
   - **Description**: `CRA临床研究助理管理中心`
   - 选择 **Public**（免费用户只能选Public）
   - ✅ 勾选 **"Add a README file"**（可选）
4. 点击 **"Create repository"**

**完成后，复制仓库URL**，例如：
```
https://github.com/你的用户名/cra-portal.git
```

---

### 第2步：推送代码到GitHub（1分钟）

打开**终端（Terminal）**，执行以下命令：

```bash
# 进入项目目录
cd ~/.qclaw/workspace/cra-portal

# 添加远程仓库（替换成你的仓库URL）
git remote add origin https://github.com/你的用户名/cra-portal.git

# 推送代码到GitHub
git branch -M main
git push -u origin main
```

**首次推送需要输入GitHub用户名和密码**（或使用Token）。

**推送成功后，刷新GitHub仓库页面，应该能看到所有代码文件。**

---

### 第3步：部署到Render（2分钟）

1. 访问 [Render.com](https://render.com) 并注册/登录
2. 点击 **"New +"** → **"Web Service"**
3. 选择 **"Connect a repository"**
4. 授权Render访问你的GitHub账户
5. 选择 `cra-portal` 仓库
6. 填写配置：
   - **Name**: `cra-portal`（或你喜欢的名字）
   - **Environment**: `Python 3`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python3 app.py`
   - **Plan**: 选择 **Free**（免费）
7. 点击 **"Create Web Service"**

**等待部署完成（约2-3分钟）**，部署成功后，Render会提供一个公网地址：

```
https://cra-portal.onrender.com
```

---

### 第4步：访问网站（1分钟）

1. 在浏览器（或手机）访问Render提供的地址
2. 你应该能看到CRA Portal主页
3. 测试功能：
   - ✅ 点击"项目"查看3142iv、0197
   - ✅ 点击项目卡片查看详情
   - ✅ 点击"编辑"测试在线编辑

## 🔧 常见问题

### Q1: 部署后访问显示"Application Error"？
**A**: 检查Render日志（Dashboard → 你的服务 → Logs），常见原因：
- `requirements.txt` 缺少依赖
- `Start Command` 写错
- 端口配置错误（Render需要监听 `$PORT` 环境变量）

**修复方法**：修改 `app.py`，添加端口配置：
```python
import os
port = int(os.environ.get("PORT", 8080))
app.run(host='0.0.0.0', port=port)
```

### Q2: GitHub推送失败？
**A**: 可能需要使用 **Personal Access Token**：
1. GitHub → Settings → Developer settings → Personal access tokens → Generate new token
2. 推送时使用token作为密码

### Q3: Render免费版会休眠？
**A**: 是的，免费版15分钟无访问会自动休眠，下次访问需要等待10-30秒唤醒。
